### Project setup

1. Clone the repo
   ```sh
   git clone https://github.com/belka-mironova/movies-explorer-frontend.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the frontend part of the application in a development mode 
   ```sh
   npm run start
   ```
